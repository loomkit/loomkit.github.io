## Quick Start

<Tabs>
  <TabItem label="New Project" icon="rocket">
```sh
composer create-project loomkit/starter my-loom-app
cd my-loom-app
bun install
php artisan migrate --seed
composer dev
```
  </TabItem>

  <TabItem label="Existing Project" icon="laptop">
```sh
composer require loomkit/loom -W
php artisan loom:install
composer dev
```
  </TabItem>
</Tabs>
